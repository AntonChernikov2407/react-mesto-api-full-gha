# Проект Mesto фронтенд + бэкенд


Сервер проекта Место с настроенными роутами и с двумя схемами для пользователей и карточек.
Также выполнена обработка основых ошибок.
Стэк примененных технологий:
1. Node.js
2. Express
3. MongoDB
4. Mongoose
